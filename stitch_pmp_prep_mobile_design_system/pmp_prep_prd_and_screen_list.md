# PMP Prep - Product Requirements & Screen List

## Overview
A minimalist Android quiz app for PMP study.
- **Design System**: One UI structure (bottom actions, large headers) + Duolingo interaction (question cards, check button).
- **Themes**: Light & Dark (System-driven).

## Screen List (Light & Dark)

1. **Home**: Collapsing "Good Morning" title, Streak/Daily Status card, Entry buttons (Daily, Quiz, Free), Bottom banner ad.
2. **Question Component (States)**: Progress bar, question text, 4 answer cards. Show Default, Selected, Correct, Incorrect states.
3. **Daily Question**: Flow with question, explanation panel, and "Done" streak confirmation.
4. **Quiz Setup**: Length selector (10/25/50) or Full Mock Exam (180 Q).
5. **Quiz In-Progress**: Question component + visible countdown timer.
6. **Quiz Results**: Score %, Pass/Fail status, Domain breakdown (bars), Time vs Target, Review list.
7. **Free Mode**: Domain filter selection, immediate feedback loop.
8. **Settings**: Reminder toggle, time picker, theme note, About.
9. **Notification**: Android-style notification mockup.
10. **States & Welcome**: Loading screen, Empty state, and First-run welcome.

## Motion & Interaction
- Smooth sliding explanation panel.
- Haptic-style feedback on correct/incorrect.
- Header collapses on scroll.
- Satisfying progress bar filling.
